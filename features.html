Features

Larry the Engine

Class Description

GameTime
keeps track of time elapsed in game so that engine updates independently of frame rate.

FrameTimer
counts down for every frame from a given count.

GameTimer
counts down for every in-game millisecond from a given count.

Texture
stores an image, along with image’s width, height and source.

Sprite
keeps an image, position, width and height. Handles collisions with points and other sprites.

SpriteSheet
keeps multiple images stored in a single texture, keeps current sprite.

Actor
keeps its position.

ActorSprite
(extends Actor) keeps position and sprite.

GamePiece
(extends ActorSprite) keeps position, sprite, speed, direction, orientation, rotation, collisions, viewability, boundaries and timers of a game object.

Stage
keeps arrays of Actors, FrameTimers, drawables and ParticleSystems. Keeps camera variable.

Particle
contains actor, attributes (speed, direction, rotation).

ParticleSystem
stores particle spawning point and array of particles. Functions include spawning and destroying particles.

Keyboard
keeps a buffer of keyboard inputs, keeps track of last keycode pressed, keeps track of whether a key is currently pressed.

Keyboard2
keeps a buffer of keyboard inputs.

Mouse
keeps track of last position of click and whether mouse button is currently pressed.

Point
keeps x and y positions.

Line
vector that keeps the start and end points.

Box
keeps position, width, height and checks for containment of sprite.

Boundary
keeps an array of lines and keeps track of collisions involving those lines.

SquareGrid
contains grid size, interval of positions within grid and obstacles inside of the grid.

breadthFirstSearch()
method of traversing grid which looks at nearest nodes first when searching for a path to a given end point from a given starting point.

Menu
stores sprite for menu, the number of possible selections and current selection.

Queue
puts newest input at the end of its buffer, removes oldest input from the front of the buffer.

Text
stores string, position, fillstyle and font.
